#### JS的变量类型

数值类型：number

字符串类型：String

对象类型：object

布尔类型：boolean

函数类型：function



#### JS中特殊的值

undefined   未定义，JS变量没有赋于初始值时，默认值都是undefined

null    空值

NAN    非数字，非数值



```html
<head>
        <meta charset="UTF-8">
        <title>Document</title>
        <script type="text/javascript">
            var i;
            alert(i);      //undefined
            i = 12;
            alert(typeof(i));    //number
        </script>
    </head>
```
