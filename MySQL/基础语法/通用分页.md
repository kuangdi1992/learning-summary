# 通用分页

每页显示3条记录：

```
第一页：limit 0,3 [0,1,2]
第二页：limit 3,3 [3,4,5]
第三页：limit 6,3 [6,7,8]
```

每页显示pageSize条记录，那么第pageNo页：

```sql
limit (pageNo - 1) * pageSize, pageSize
```

```sql
public static void main(String[] args){
    int pageNo = 5;
    int pageSize = 10;
    
    int startIndex = (pageNo - 1) * pageSize;
    String sql = "select ... limit " + startIndex + "," + pageSize;
}
```
